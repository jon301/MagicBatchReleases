Public releases of the software MagicBatch

## Downloads

### macOS
- **Apple Silicon (M1/M2/M3)**: Download the `aarch64.dmg` file
- **Intel**: Download the `x64.dmg` file

### Windows
- Download the `.msi` installer

## Installation

### macOS
1. Download the appropriate DMG file for your Mac
2. Open the DMG file
3. Drag MagicBatch to your Applications folder
4. On first launch, you may need to right-click and select "Open" due to macOS security

### Windows
1. Download the MSI installer
2. Run the installer
3. Follow the installation wizard
